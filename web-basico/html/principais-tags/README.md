# Principais TAG's

Os próximos tópicos irão mostrar as princinpais TAG que são utilizadas. Você pode conferir a lista completa de TAG que existem em: [https://www.w3schools.com/tags/](https://www.w3schools.com/tags/)

## Sumário

* Formatação
  * &lt;h1 até h6&gt;
  * &lt;br&gt;
  * &lt;p&gt;
* Básicas
  * &lt;html&gt;
  * &lt;head&gt;
  * &lt;body&gt;
  * &lt;title&gt;
  * &lt;!-- --&gt;
  * &lt;!DOCTYPE html&gt; 
* Entrada de Dados
  * &lt;input&gt;
  * &lt;button&gt;
* Listas
  * &lt;ol&gt;
  * &lt;ul&gt;
  * &lt;li&gt;
* Link
  * &lt;a&gt;
* Organização
  * &lt;div&gt;
  * &lt;span&gt;
* Multimídia
  * &lt;img&gt;
